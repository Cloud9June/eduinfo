S:NOW 학생용 웹앱
S:NOW는 성일정보고 학생들의 학교생활 편의를 위해 개발된 통합형 웹앱입니다. 급식 확인, 피드(공지) 확인, 대외활동 안내, 디지털 시민교육 콘텐츠 등 다양한 기능을 한 화면에서 사용할 수 있도록 설계되었습니다. Firebase를 기반으로 교사·학생 권한에 따라 화면이 다르게 구성되며, PWA(Progressive Web App) 형태로 개발되어 스마트폰에 앱처럼 설치할 수 있습니다. 또한 키오스크 모드로도 실행 가능하도록 최적화되어, 학교 내 정보 안내용 대형 화면에서도 동일하게 작동합니다.

문제 해결 사례
Firebase 보안 규칙 개선
초기에는 localStorage를 통한 권한 관리로 보안 위험이 있었으나, Firestore 기반 users 컬렉션과 Security Rules를 적용하여 관리자·담임·학생 권한을 세분화했습니다.
앱 설치 UX 개선
키오스크용 Chrome 환경과 모바일 브라우저의 차이로 설치 안내 혼란이 발생하여, ‘도움말’ 탭에 QR코드 및 앱 설치 튜토리얼을 추가했습니다.
성과 및 발전 방향
현재 개발 사항
현재 학교 내에서 실제 운영 중이며, 교사·학부모 계정 통합과 푸시 알림 기능 추가를 목표로 버전 4.0을 개발 중입니다.
